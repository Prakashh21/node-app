REST API

GET /api/users -- List all users -- JSON

GET api/users/1 -- get the data of user with id = 1 --JSON
GET api/users/2 -- get the data of user with id = 2

GET /user/1 -- sends the server side rendered page with data of the user



POST /api/users - Create new user

PATCH /api/users/id/1 - Edit the user with ID 1

DELETE /api/users/id/1 - Delete the user with ID 1
